# Castles  
Castles is a GameEngine that uses C# and Giawa's OpenGl Library.  

## Features
- Entity rendering  
- Terrain generation  
- Diffuse mapping  
- Normal mapping  
- Skybox rendering  
- Fog/Fadeout  
- Water rendering including diffraction and reflection
